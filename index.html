What is the Xenlism project?

Xenlism is a computer graphic and programming project to make something better.

Xenlism minimalism is a minimalistic Gnome/GTK Theme

Inspired by Elementary OS and based on the Arc GTK Theme.

Xenlism wildfire is an icon theme for *nix desktop environments.

Inspired by Nokia's Meego and Apple iOS's icon theme.

Great for Gnome and (maybe) Unity, Mate,Cinnamon.
